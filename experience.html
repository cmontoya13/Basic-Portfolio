Experience

Working Skills

My foremost goals when developing a website are user experience and search engine optimization. Making sure that all users are able to view and interact flawlessly with a website is a balancing act of many variables. I minimize restrictive scripts by coding as much as possible with HTML5 and CSS3. Where scripting is used, I always provide an alternative path for users who deny scripts. Onload speed is increased by decreasing server requests, moving scripts to the page bottom or setting them for asynchronous load. I use keyword optimization tools to find the best words and phrases to incorporate within the page title, headers and content without over saturating. Stickiness is achieved by utilizing insightful content, HTML5 effects like transitions and animations, interesting imagery, complimentary colors and JavaScipt functions. I hold several #1 positions in Google.
